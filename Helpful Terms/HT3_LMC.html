Helpful Terms

  • HTTP - an abbreviation for Hyper Text Transfer Protocol, HTTP is is the mode by which our devices request servers to view a webpage; in other words, "HTTP is the language your web browser uses to speak with the web server in order to inform it of a request" (source: Ionos: What is HTTP?)

  • Internet Backbone - the internet backbone is the network of submarine cables that provide us with internet access. The way it functions is by sending information through high-speed access points that route it back to our personal computers (source: Network World: Internet Backbone)

  • IP Address - an IP address is essentially our computerâ€™s home address. IP addresses are unique to the devices being used to access the internet

  • ISP - ISP is an acronym for Internet Service Providers. The ISP is the company we pay to be provided with internet access. Examples of current ISPs are AT&T, Charter/Spectrum, TDS, Google Fiber, and more. We can also think of our wireless plan providers as ISPs

  • Javascript - Javascript is a front-end computer programming language that dictates the behavior of a website. Javascript allows for dynamic web pages, by prescribing actions to variables, var, existing elements created through HTML & CSS.

  • Objects - Objects are variables. Almost all JavaScript values, except primitives, are objects

  • Primitive Data Types - in JavaScript, denotes the data that is not an object and has no methods. Six primitive data types include string, number, bigint, boolean, undefined, and symbol

  • Project Management - is the process of leading a team in the completion of specific IT projects or tasks

  • Strings - JavaScript's String type is used to represent textual data. It is a set of "elements" of 16-bit unsigned integer values. Each element in the String occupies a position in the String. The first element is at index 0, the next at index 1, and so on. The length of a String is the number of elements in it.

  • Waterfall - project management framework in which activities are broken down into linear sequential phases. Often, each phase depends on the deliverables of the previous; using this framework, all aspects of a project are predetermined before a final launch, which may result in having to start from the beginning if changes are required (source: Wikipedia: Waterfall model)

  • Web Server - the web server stores the webpage files; the DNS server is what connects users to these sites (source: Medium: Domain Registrar or DNS Hosting: The Difference Explained)
